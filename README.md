## Опис проєкту
У межах цієї роботи було створено мінісайт із трьох сторінок, що демонструє володіння основами HTML5 та CSS:
- **Семантична розмітка:** використання тегів header, nav, main, footer, figure[cite: 161, 166].
- **Контент:** робота із заголовками, списками, таблицями та текстом [cite: 163-167].
- **Форми:** реалізовано форму зворотного зв'язку з валідацією полів [cite: 172-181].
- **Медіа:** вбудовані аудіо- та відеофайли з елементами керування [cite: 186-190].
- **Стилізація:** додано зовнішній CSS файл для сучасного вигляду та адаптивності.

## Як запустити
