## Опис проєкту
У межах цієї роботи було створено мінісайт із трьох сторінок, що демонструє володіння основами HTML5 та CSS:
- **Семантична розмітка:** використання тегів header, nav, main, footer, figure.
- **Контент:** робота із заголовками, списками, таблицями та текстом.
- **Форми:** реалізовано форму зворотного зв'язку з валідацією полів. 
- **Медіа:** вбудовані аудіо- та відеофайли з елементами керування.
- **Стилізація:** додано зовнішній CSS файл для сучасного вигляду та адаптивності.

## Як запустити
1. Склонуйте репозиторій.
2. Відкрийте файл \`index.html\` у будь-якому сучасному браузері або скористайтеся розширенням **Live Server** у VS Code.
